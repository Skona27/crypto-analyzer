# crypto-analyzer

# Pre-requisites

- Install [Node.js](https://nodejs.org/en/) version 18.0.0

# Getting started

- Clone the repository

```
git clone  crypto-analyzer
```

- Install dependencies

```
cd crypto-analyzer
npm install
```

- Build and run the project

```
npm run build
npm start
```

See the output in the terminal.

- Run unit tests

```
npm run test
```
